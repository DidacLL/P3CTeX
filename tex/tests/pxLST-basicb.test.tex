\section*{Exemple bàsic de pxLST}

\pxCodeSet[dark]{pxJava}[pxlst]

El codi següent mostra un \emph{hello world} senzill en Java:

\begin{pxCode}[language=pxJava]{HelloWorld.java}
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hola, món!");
  }
}
\end{pxCode}

També podem inserir fragments en línia amb \pxCodeIn[pxJava]{suma(a,b)}.
